# RPGItems
A Minecraft Item Management plugin for RPG Servers

Dependencies:
- [ProtocolLib](https://www.spigotmc.org/resources/protocollib.1997/)
- [Eco](https://github.com/Auxilor/eco)
- Spigot 1.19+

Commands:
- rpgitems give itemName amount userName
- rpgitems reload

Permissions:
- rpgitems.admin
